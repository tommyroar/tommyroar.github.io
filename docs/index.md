## Featured Projects

### [Vitamind](/maps/vitamind/)
**Vitamind** is a specialized tool designed to help users estimate their Vitamin D production from sunlight. By combining geographic data (via Mapbox) with solar position calculations (via SunCalc), it provides insights into the best times and locations for natural Vitamin D synthesis.

- **Status:** Active
- **Tech Stack:** React, Vite, Mapbox GL JS, SunCalc
- **Access:** [Open Vitamind App](/maps/vitamind/)
